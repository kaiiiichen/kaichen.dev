# Contributing

This repository is a personal site. External contributions are welcome for clear bugs or small improvements; large redesigns are best discussed first.

## Workflow

1. Branch from `main` (e.g. `feat/short-description`).
2. Run `npm run lint` and `npm run build` before opening a PR.
3. Describe what changed and any new or required environment variables.

## Style

- Match existing TypeScript, Tailwind, and component patterns in `app/components/`.
- Prefer focused diffs; avoid unrelated refactors in the same PR.
